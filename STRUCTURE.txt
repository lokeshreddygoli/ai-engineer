File Statistics
===============
Total Python code:     ~1,340 lines
Total documentation:   ~1,500 lines
Total files:           ~20 files

Key Files
=========
- src/app.py              : FastAPI service with 4 endpoints
- src/vector_store.py     : Sentence-transformers embeddings
- src/fuzzy_clustering.py : GMM with BIC cluster selection
- src/semantic_cache.py   : Custom similarity-based cache
- Dockerfile              : Production-ready container
- docker-compose.yml      : One-command deployment
- README.md               : Complete system documentation

Quick Start
===========
# Using Docker (recommended)
docker-compose up --build

# Using Python
bash setup_venv.sh
source venv/Scripts/activate
uvicorn src.app:app --reload

API Documentation
=================
http://localhost:8000/docs       # Swagger UI
http://localhost:8000/redoc      # ReDoc
http://localhost:8000/health     # Health check

Endpoints
=========
POST   /query          # Semantic search with caching
GET    /cache/stats    # Cache performance metrics
DELETE /cache          # Clear cache
GET    /health         # System status
